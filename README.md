# plusコマンド
![test](https://github.com/Nakayma-Eiki/robosys2025/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ数字を足す。

## 使い方例
$git clone git@github.com:Nakayama-Eiki/robosys2025.git

$seq 5 | ./plus

15

## 必要なソフトウェア
- Python
  - テスト済みバージョン: 3.7〜3.10

## テスト環境
- Ubuntu 24.04 LTS
